# DataJam-Nov-2025






### Setup 
1. `git clone https://github.com/Abyss999/DataJam-Nov-2025.git`
2. Activate virtual environment 
3. Install requirements 
- `pip install -r requirements.txt`
### Virtual Environment 
python3 -m venv datajam
Windows `datajam\Scripts\activate`
Mac: `source datajam/bin/activate`
